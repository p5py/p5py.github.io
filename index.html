p5

p5 is a Python package based on the core ideas of Processing. It aims to make programming more accessible for beginners, artists, and educators by combining Python's readability with Processing's emphasis on learning to code in a visual context.

p5 is being developed by Abhik Pal as a project for the Google Summer of Code. Manindra Moharana of the Processing Foundation is mentoring the project.

See the project repo for details.
